# ☕ Core Java — JSpider Bengaluru

> My Core Java practice code, assignments, and exercises from **JSpider Training Institute, Bengaluru**.  
> This repo tracks my learning journey through Core Java concepts from the ground up.

---

## 🏫 About

This repository contains all the Java programs, assignments, and hands-on exercises I wrote during my **Core Java training at JSpider, Bengaluru**. It covers foundational to intermediate Java concepts practiced under guided instruction.

---

## 📚 Topics Covered

- ☑️ Java Basics — Variables, Data Types, Operators, Control Flow
- ☑️ Object-Oriented Programming — Classes, Objects, Constructors
- ☑️ OOP Pillars — Inheritance, Polymorphism, Abstraction, Encapsulation
- ☑️ Interfaces & Abstract Classes
- ☑️ Packages & Access Modifiers
- ☑️ Exception Handling — try/catch/finally, Custom Exceptions
- ☑️ Arrays & Strings
- ☑️ Collections Framework — ArrayList, HashMap, HashSet, LinkedList
- ☑️ Generics
- ☑️ File I/O — FileReader, FileWriter, BufferedReader
- ☑️ Multithreading — Thread, Runnable, Synchronization
- ☑️ Java 8+ Features — Lambda Expressions, Stream API, Functional Interfaces
